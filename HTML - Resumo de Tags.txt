<!DOCTYPE html> -> formato (html 5)

<html> -> informa que o conteúdo é html / não nula

<head> -> cabeçalho (meta e title) / não nula

<body> -> conteúdo para o usuário final / não nula

<title> -> título da página / não nula

<hn> -> <h + nível desejado> Título </h + nível desejado> / não nula

<br> -> quebra de linha / nula

<meta> -> indicar a codificação (<meta charset="utf-8">> 

<i> -> itálico - nomes técnicos / não nula

<em> -> itálico - ênfase / não nula 

<u> -> sublinhado / não nula

<sub> -> subscrito / não nula

<sup> -> sobrescrito / não nula

<strong> -> negrito - importante / não nula

<b> -> negrito / não nula

<p> -> paragráfo / não nula

<hr> -> linha horizonal / nula

<a href = "destino" target = "_blank"> texto do link -> inserção de links / não nula 

<font face = "estilo"> texto -> estilo da fonte / não nula

<font size = "tamanho da fontepx"> -> tamanho da fonte do texto / não nula

<font color = "cor"> texto -> cor do texto / não nula

<body bgcolor = "cor"> -> obs: dentro da tag body - cor background / não nula

<ul> -> lista não ordenada / não nula

<ol> -> lista ordenada / não nula

<li> -> cada item da lista / não nula

<dl> -> lista de termos e descrições / não nula

<dt> -> termos da lista / não nula

<dd> -> descrição dos termos / não nula

<img src = "Nome do Arquivo. formato" width = "tamanho"> -> imagem / nula

<table> ou <table border = 'tamanho da borda' -> tabela com borda e sem borda / não nula

<tr> -> linhas da tabela / não nula

<td> -> colunas da tabela / não nula

<form action = "Destino dos dados (banco de dados)" method = "método do HTTP (get ou post)"> -> criação de formulário / não nula

<label> Texto -> textos do formulário / não nula

<input type = "tipo (text ou password)"> -> campo de texto e campo de senha (..........) de uma linha / nula

<textarea> </textarea> ->  campo de texto maior e redimensionável / não nula

<input type = "hidden" name = "nome" value = "valor"> -> campo invisível para o usuário / nula

<button type- "Tipo do botão (submit, reset ou button)"> Texto do botão -> botão do formulário / não nula

<input type = "checkbox" name = "nome" value = "valor" checked (com ou sem)> Texto -> opções de múltipla escolha / nula 

<input type = "radio" name = "nome" value = "valor" checked (com ou sem)> Texto -> única escolha / nula

<select> -> lista suspensa (drop - down) / não nula

<option value = "valor"> Texto -> cada item da lista suspensa / não nula
